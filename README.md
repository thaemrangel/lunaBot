# Luna Bot


### What is Luna Bot?
Luna bot is a small discord bot made using Discord.py library, which is a library that is written in Python.  

Since I wanted to learn more about
Python, and also make a rpg status Bot for me and my friends, I made Luna Bot.


### How to run?

You need to create your own bot via [Discord's Developer Portal](https://discord.com/developers/applications).  When you have your token, and after, when downloading this project, 
create a .env file.  In the .env file, configure global tokens as so:

```
DISCORD_BOT_TOKEN='your cool token here'
AUTHORIZED_PLAYER_1='your id'
AUTHORIZED_PLAYER_2='your friend's id'
```


This project is supposed to be a small 'boilerplate-ish' of a bot.

### What Luna can do?

#### Currently, Luna bot can:

- Give you specific information about a player.
- Give you all information about a player.
- Update information about a player (if you have permission).
- Say hello back to you! :D

Luna Bot also have a list of players as a JSON. 
I didn't had no need to add a command to add players, however, 
since I want to update this bot, it will have in future versions when
I migrate Luna to a DB.  For how, that's it.
